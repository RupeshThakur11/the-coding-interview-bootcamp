<span className='instructions'>
  Write a function that returns the number of vowels used in a string. Vowels
  are the characters 'a', 'e' 'i', 'o', and 'u'.
</span>

<div className='instructions example'>Examples</div>

<div className='instructions'>vowels('Hi There!') --> 3</div>
<div className='instructions'>vowels('Why do you ask?') --> 4 </div>
<div className='instructions'>vowels('Why?') --> 0</div>

```
// first (ðŸŒ¶)

function vowels(str) {
	let count = 0;
	const checker = ['a', 'e', 'i', 'o', 'u'];

	for (let char of str.toLowerCase()) {
		if (checker.includes(char)) {
			count++;
		}
	}
	return count;
}
```
