_Given a string, return true if the string is a palindrome or false if it is not._
_Palindromes are strings that form the same word if it is reversed._
_Do include spaces and punctuation in determining if the string is a palindrome._

> _Two differents ways_

```
// first (🍒)
function palindrome(str) {
  const reversed = str
    .split('')
    .reverse()
    .join('')
  return reversed === str;
  // should return true or false
}

// Second (🍑)
function palindrome(str) {
  return str.split('').every((char, i) => {
    return char === str[str.length - i - 1];
  });
}
```
