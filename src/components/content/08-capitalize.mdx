<span className="instructions">
  Write a function that accepts a string. The function should capitalize the
  first letter of each word in the string then return the capitalized string.
</span>

<p className='instructions example'>Examples</p>

<p className='instructions'>- capitalize('a short sentence') --> 'A Short Sentence'</p>
<p className='instructions'>- capitalize('a lazy fox') --> 'A Lazy Fox'</p>
<p className='instructions'>- capitalize('look, it is working!') --> 'Look, It Is Working!'</p>

> <span className="ways">Two differents ways</span>

```
// First (🍅)
function capitalize(str) {
  const words = [];

  for (let word of str.split(' ')) {
    words.push(word[0].toUpperCase() + word.slice(1));
  }
  return words;
}
// Second (🍆)
```
