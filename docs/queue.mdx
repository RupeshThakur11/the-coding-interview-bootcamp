# Queue

_Instructions:_

- Create a queue data structure. The queue should be a class with methods 'add'
  and 'remove'. Adding to the queue should store an element until it is removed

_Examples:_

```shell
const q = new Queue();
q.add(1);
q.remove(); // returns 1;
```

_Repl.it:_

- <a href='https://repl.it/@rojasleon/14-queue' target='_blank'>queue</a>

_Code:_

```javascript
class Queue {
  constructor() {
    this.data = [];
  }

  add(record) {
    this.data.unshift(record);
  }

  remove() {
    return this.data.pop();
  }
}
```
