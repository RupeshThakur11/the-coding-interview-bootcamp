# Max Char

_Instructions:_

- Given a string, return the character that is most commonly used in the string.

_Examples:_

- `maxChar("abcccccccd") === "c"`
- `maxChar("apple 1231111") === "1"`

_Repl.it:_

- <a href='https://repl.it/@rojasleon/04-max-char' target='_blank'>max-char</a>

_Code:_

```javascript
function maxChar(str) {
  let charMap = {};
  let max = 0;
  let maxChar = '';

  for (let char of str) {
    charMap[char] = character[char] + 1 || 1;
  }

  for (let char in charMap) {
    if (charMap[char] > max) {
      max = charMap[char];
      maxChar = char;
    }
  }
  return maxChar;
}
```
