# Reverse an Int

_Instructions:_

- Given an integer, return an integer that is the reverse ordering of numbers.

_Examples:_

- `reverseInt(15) === 51`

- `reverseInt(981) === 189`

- `reverseInt(500) === 5`

- `reverseInt(-15) === -51`

- `reverseInt(-90) === -9`

_Repl.it:_

- <a href='https://repl.it/@rojasleon/03-reverse-an-int' target='_blank'>reverse_an_int</a>

_Code:_

```javascript
function reverseInt(int) {
  const reversed = int
    .toString()
    .split('')
    .reverse()
    .join('');

  // Math.sing returns
  // 1 if (int) is positive
  // -1 if is negative
  // 0 if is 0
  return parseInt(reversed) * Math.sign(int);
}
```
